# Centipede

A recreation of the 1981 Atari "Centipede" game using HTML Canvas with a focus on OOP.  Additionally, it was my first project built with ES6 and TDD.

## Tools and Technologies

* HTML5 Canvas
* ES6
* Mocha/Chai
